by Jiani

2019-02-21

Preface

Bookdown R package is a simple way to publish a GitBook look-and-feel online book and turn it into a static wiki site in no time.

This is a simple minimum book to test how this works.

I used a number of sources during this test:

  • R package Bookdown by Yihui, RStudio
  • Tidyverse Style Guide by Hadley Wickham
  • Bookdown Starter “Boilerplate” by Sean Kross
  • R Markdown
  • Markdown guide
  • Lorem Ipsum generator

Sean’s Bookdown starter boilerplate has proven to be invaluable to kick start this simple book.
